# Monkey Market Clone

A production-ready, high-performance web clone of the "Monkey Mart" idle genre, built using Vanilla JavaScript and HTML5 Canvas.

## Features
- **Smooth Gameplay:** Optimized 60FPS physics and movement.
- **Save System:** Progress is automatically saved to LocalStorage.
- **Upgrades:** Spend bananas (money) to increase speed and carry capacity.
- **Procedural Assets:** No external image files required.
- **Audio:** Real-time synthesized sound effects using Web Audio API.

## Controls
- **WASD or Arrow Keys:** Move the Monkey.
- **Mouse/Touch:** Click buttons in UI.
- **Automatic:** Stand near the Green Zone (Top-Left) to pick up bananas. Stand near shelves to stock them.

## Installation / Hosting
1.  Copy all files into a single directory.
2.  Open `index.html` in any modern web browser.
3.  **To Host on GitHub Pages:**
    - Create a new repository.
    - Upload `index.html`, `style.css`, and `main.js`.
    - Go to Settings > Pages and select the `main` branch.
